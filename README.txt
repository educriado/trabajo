Para la ejecucion del programa basta con la siguiente orden en la linea de
comandos:
$ python trabajo.py

Si queremos cambiar el uso de bigramas y/o la distribucion con la que aprende
el clasificador tenemos dos variables, clasificador (linea 239) y uso_bigrama
(linea 240). La primera admite los valores "Multinomial" y "Bernoulli" y la
segunda True o False.
